## Conventions (recommended)

### PDF filenames
Use:
- `AuthorYYYY - Short Title.pdf`

Examples:
- `Duflo2006 - Field Experiments in Development Economics.pdf`
- `Banerjee2011 - Poor Economics (Ch1).pdf`

### Notes style
Keep notes lightweight and skimmable:
- key takeaways (bullets)
- identification / design notes (if empirical)
- what to follow up on (citations to find, replications, related papers)

### Notebooks
- Put notebooks in the week’s `code/` folder.
- Ignore Jupyter checkpoints via `.gitignore`:
  - `.ipynb_checkpoints/`
